About

This simplistic version of the classic Atari game Breakout was made for the chipKIT PIC32 microcontroller, with the Basic I/O Shield that boasts a 128x32 px OLED display. By rewriting the hardware dependent code it has been ported to the web using WebAssembly.

Play against a friend, or face the cheating AI opponent that unfairly tracks the speed of the ball.
